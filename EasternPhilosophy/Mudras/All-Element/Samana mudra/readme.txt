Method:each hand fingertips brought together. Hand apart.
Effect:Balance. Useful for emptiness meditation, targeted healing(move hand towards area), recovery and calm: balances all elemental flows.
Additional:
For increase/release of Yin energy, use only the left hand for mudra/targeted healing.
For increase/release of Yang energy, use only the right hand for mudra/targeted healing.
